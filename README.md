# PrimaryCalculator

这是一个Java作业，写了一个初级计算器，本来算法是使用Java写的，但是后续如果要添加按键的话就会非常麻烦，这里选择调用Python来进行计算
UI是纯Java做的；计算利用Python可以直接运行字符串特点进行计算

#项目结构

```bash
├───src
│   ├───hy
│   |   ├───java
│   |   |   ├───GradientButtonTextUI
│   |   |   ├───MyPane

│   |   |   └───RoundedBorder
|   |   ├───Resources
│   |   |   ├───Image
│   |   |   |   ├───March 7 verticval screen version
│   |   |   └───Python37
```
